# Project Structure

The project is organized into the following directories and files:

- **src/**: Contains all the source code for the project.
  - **components/**: Contains reusable components.
  - **pages/**: Contains the main pages for the application.
  - **styles/**: Contains stylesheets for the project.

- **tests/**: Contains unit and integration tests.

- **docs/**: Contains documentation files.

- **README.md**: This file contains an overview of the project and its structure.

- **package.json**: Details about project dependencies and scripts.
